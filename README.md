# SEC Form 4 Fetcher

Fetches and parses SEC Form 4 filings (insider transaction reports) from EDGAR.

## Features

- Fetch recent Form 4 filings via EDGAR RSS feed
- Parse XML filings to extract insider transaction details
- Identify transaction type (buy/sell), shares, price, and ownership after
- Extract insider information (name, title, officer/director status)
